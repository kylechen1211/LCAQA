# LCAQA
輕量且抗量子攻擊之通訊行動應用程式

## 簡介
* 建立一個以具準交換性及吸收性的雜湊函數結合對稱式加密為核心並外掛在現有通訊軟體上，並達到抗量子攻擊且高效能的目的。
